# Task2---Data-Cleaning-and-Missing-Value-Handling
This project focuses on data cleaning and missing value handling for two datasets: House Prices and Medical Appointment No Shows.

House Prices Dataset:-

Source: Kaggle

Handled missing values

Median imputation for numerical features

Mode imputation for categorical features

Dropped columns with very high missing percentages

Exported cleaned file: house_prices_cleaned.csv

Medical Appointment No Shows Dataset:-

Source: Kaggle

Introduced missing values in Age and Neighbourhood for imputation practice

Applied median imputation for Age

Applied mode imputation for Neighbourhood

Exported cleaned file: medical_appointment_cleaned.csv

Tools Used:-

Python

Pandas

NumPy

Matplotlib

Seaborn

Outcome:-

Both datasets were successfully cleaned, all missing values were handled, and the final files are ready for further analysis or machine learning tasks.
